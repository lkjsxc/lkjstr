# Styles

## Purpose

This directory contains static CSS loaded by the Rust/WASM browser entrypoint.

## Table of Contents

- `tabs.css`: tab body, Welcome, and New Tab menu styles.
- `relay-settings.css`: Rust Relay Settings tool styles.
- `ui.css`: base browser app styling.
- `upload-settings.css`: Rust Upload Settings tool styles.
- `workspace.css`: workspace shell, pane, and tab strip styles.
