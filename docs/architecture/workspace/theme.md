# Theme

## Contract

The default theme is dark, mostly achromatic, and low radius.

## Tokens

- Backgrounds are near black neutral surfaces.
- Text uses high-contrast neutral values.
- Borders and surface contrast indicate active UI state.
- Color is reserved for semantic danger, warning, success, and focus states.
- General controls use radius values from zero through two pixels.
- Avatars may remain circular.

## Behavior

Theme and radius settings apply through CSS variables. Active tabs use contrast,
not saturated color. Resize handles are neutral, visible, and square.
