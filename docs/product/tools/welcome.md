# Welcome

## Purpose

Welcome gives startup status for accounts, relays, and the core workspace
tabs.

## Contract

- Welcome is the clean first-launch tab and can also open from New Tab.
- Storage failure recovery falls back to a usable Welcome workspace.
- The tab shows active account, account count, selected relay set, read relay
  count, and write relay count.
- If no active account exists, Welcome directs attention to Accounts.
- If no read relays are enabled, Welcome directs attention to Relay Settings.
- Once account and read relay prerequisites are present, Welcome reports the
  main workspace tabs as ready.
