# Workflows

## Purpose

Workflows describe common user paths through the app.

## Open Data

1. Open New Tab.
2. Select any New Tab choice: Home, Tweet, Notifications, Search, Custom
   Request, Global, My Profile (when signed in), Profile Edit, Accounts, Relay
   Settings, Stats, Settings, Upload Settings, lkjstr Log, Mine npub, or
   Welcome.
3. Use identity and event actions to open Profile or Thread tabs.

## Tab Drag

1. On touch, pan the tab rail horizontally to reach hidden tabs.
2. Long-press a tab to arm drag, then move to reorder within a tile, move to
   another tile center, or drop on a tile edge to split.
3. On mouse, press and move a tab after the activation threshold.
4. Reorder along the tab strip takes precedence over accidental edge splits
   while the pointer remains in the source strip band.

## Publish

1. Add a NIP-07 account.
2. Enable at least one write relay in the selected default set.
3. Open Tweet.
4. Write content and publish through the browser signer.
