# Privacy

## Purpose

This subtree defines the user-visible privacy and consent surfaces.

## Table of Contents

- [consent.md](consent.md): first-run banner, consent choices, and optional
  categories.
- [settings.md](settings.md): privacy settings, withdrawal, and optional-data
  clearing.

## Contract

lkjstr is local-first. Essential browser storage remains available for the
workspace, accounts, relay settings, drafts, cache, diagnostics, and consent
record because those features are the product. Optional cookies, optional
telemetry, and optional non-essential storage stay disabled until explicit
consent.
