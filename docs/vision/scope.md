# Scope

deployment model yet.

- Vision notes.

## Out Of Scope Now

- Product requirements.
- Protocol design.
- Architecture decisions.
- Operations procedures.
- Research notes.
- Application source.
- Tests and scripts.

Those areas should be created only when there is concrete work to record.
