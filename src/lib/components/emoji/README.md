# Emoji Components

## Purpose

This directory contains shared UI for Unicode and NIP-30 custom emoji
selection.

## Table of Contents

| File                        | Purpose                                                |
| --------------------------- | ------------------------------------------------------ |
| `EmojiPaletteButton.svelte` | Anchored button that opens the shared emoji picker.    |
| `EmojiPopover.svelte`       | Renders Unicode picker plus real custom emoji choices. |
