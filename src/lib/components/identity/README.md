# Identity Components

## Purpose

This directory contains Svelte components for profile identity display.

## Table of Contents

- `Avatar.svelte`: deterministic avatar shell and real image rendering.
- `IdentityChip.svelte`: compact inline identity chip.
- `ProfileHoverCard.svelte`: profile hover preview.
- `FeedIdentityHeader.svelte`: leading feed row for a target pubkey.
- `UserEventRow.svelte`: feed-like reusable user row.
- `UserEventRowMeta.svelte`: user-row identity metadata.
- `UserRowOverflowMenu.svelte`: three-dot secondary actions for user rows.
- `user-row-copy-status.ts`: explicit npub copy success and failure labels.
- `user-event-row.ts`: user-row pure view helpers.
