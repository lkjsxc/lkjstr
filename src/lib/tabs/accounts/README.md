# Account Tabs

## Purpose

This directory contains account management tab components.

## Table of Contents

- `AccountManagerTab.svelte`: account add, reveal, active, disconnect, and
  copy actions.
- `AccountRow.svelte`: individual account controls.
- `AccountStorageSafety.svelte`: protected storage notice.
- `account-copy-status.ts`: explicit local secret copy success and failure
  labels.
