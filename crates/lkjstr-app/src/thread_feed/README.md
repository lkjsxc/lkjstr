# Thread Feed

## Purpose

Rust app models for Thread feed rows, context rows, and paging.

## Table of Contents

- `build.rs`: Thread feed view construction.
- `context_rows.rs`: unavailable parent and continuation row builders.
- `defaults.rs`: default Thread feed state.
- `mod.rs`: module exports.
- `paging.rs`: older-page input and cursor helpers.
- `types.rs`: Thread feed input and output records.
