# Profile Feed

## Purpose

Rust app models for Profile feed rows and cached header state.

## Table of Contents

- `build.rs`: Profile feed view construction.
- `defaults.rs`: default Profile feed state.
- `header.rs`: cached profile metadata and follow-count view data.
- `mod.rs`: module exports.
- `relay_sets_json.rs`: relay-set copy payload builder.
- `types.rs`: Profile feed input and output records.
