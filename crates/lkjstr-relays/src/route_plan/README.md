# Route Plan Source

## Purpose

Route-plan source files define pure relay route grouping for selected fallback,
author route evidence, disabled-relay exclusion, and score-based ordering.

## Table of Contents

- `mod.rs`: module exports.
- `plan.rs`: route-plan reducer.
- `plan_tail.rs`: route-plan reducer helpers.
- `types.rs`: route-plan inputs, groups, evidence, and diagnostics.
