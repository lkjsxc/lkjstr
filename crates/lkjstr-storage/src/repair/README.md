# Repair Source

## Purpose

This directory contains conservative storage repair reporting models.

## Table of Contents

- `mod.rs`: public repair exports.
- `backfill.rs`: ledger backfill planning and safe row extraction.
- `finding.rs`: stable repair finding labels.
- `probe.rs`: storage-owned physical target probe routes and output rows.
- `scan.rs`: chunked scan and inventory report records.
